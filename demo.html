Ancient Philosophy Meets Modern Diffusion 古老哲学与现代扩散模型的相遇

IChingWuxingScheduler is a novel diffusion model sampler that integrates the ancient wisdom of Wuxing (the Five Elements) with modern numerical methods, creating a self-regulating, dynamically evolving sampling system. IChingWuxingScheduler 是一款新颖的扩散模型采样器，它将五行（金木水火土）的古老智慧与现代数值方法相融合，创建了一个能自我调节、动态演化的采样系统。

Quick Start 快速开始

1. Installation 1. 安装 

pip install iching-wuxing-scheduler
# Make sure you have diffusers and torch installed
pip install diffusers transformers torch

2. Usage 2. 使用方法

IChingWuxingScheduler is a drop-in replacement compatible with the `diffusers` ecosystem. IChingWuxingScheduler 可作为直接替代品，与 `diffusers` 生态系统完美兼容。 

from diffusers import StableDiffusionPipeline
import torch
from iching_wuxing_scheduler import IChingWuxingScheduler

# 1. Load your model
model_id = "runwayml/stable-diffusion-v1-5"
pipe = StableDiffusionPipeline.from_pretrained(model_id, torch_dtype=torch.float16)

# 2. Swap the scheduler
pipe.scheduler = IChingWuxingScheduler.from_config(
    pipe.scheduler.config, mode='wuxing'
)
pipe.to("cuda")

# 3. Run the pipeline
prompt = "a masterpiece painting of a serene landscape..."
image = pipe(prompt, num_inference_steps=30).images[0]
image.save("wuxing_result.png")
